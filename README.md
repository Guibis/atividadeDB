# Amigo Fiel Pet Shop — Sistema de Gestão 🐾

Este é um projeto front-end para um sistema de gestão completo de um Pet Shop. Ele fornece uma interface moderna, amigável e responsiva para administrar o seu negócio.

A aplicação inclui painéis (dashboards) e formulários, estruturada com navegação em barra lateral (sidebar) e exibições instantâneas de dados cruciais do pet shop.

## 🚀 Funcionalidades

- **Dashboard:** Visão geral com métricas rápidas (total de clientes, pets, produtos e histórico das últimas compras).
- **Gestão de Clientes:** Listagem, busca e cadastro de novos tutores (clientes), incluindo formatação automática de CPF e telefone.
- **Gestão de Pets:** Controle da associação entre o pet e o tutor (cliente), raça e tipo do animal.
- **Gestão de Produtos:** Controle de estoque com alertas visuais integrados (para estoque baixo), indicando preço e dados do produto.
- **Registro de Compras:** Histórico completo de vendas e cálculo de totais.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído puramente com os pilares básicos da web, não necessitando de frameworks pesados para rodar:

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização vanilla, uso de CSS Variables, Flexbox, CSS Grid e media queries para interface 100% responsiva (mobile-friendly).
- **JavaScript (ES6):** Manipulação de DOM, estados simulados dos registros (CRUD) e lógica de navegação.
